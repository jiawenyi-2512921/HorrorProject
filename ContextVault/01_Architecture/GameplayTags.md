# Gameplay Tags

## Naming convention

- `Event.*` for cross-system events
- `State.*` for AI or gameplay runtime state
- `Puzzle.*` for puzzle phases
- `Archive.*` for collectible or matching semantics
- `VHS.*` for presentation state and degradation
- `Director.*` for pacing and intensity

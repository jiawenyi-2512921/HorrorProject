### 5. 构建上传

#### 5.1 使用SteamPipe上传

**安装Steam CMD：**
```bash
# 下载SteamCMD
# Windows: https://steamcdn-a.akamaihd.net/client/installer/steamcmd.zip

# 解压并运行
steamcmd.exe
```

**登录：**
```bash
login your_steam_username
```

**创建应用清单（app_build.vdf）：**
```
"AppBuild"
{
    "AppID" "YOUR_APP_ID"
    "Desc" "Build Description"
    "BuildOutput" "D:\\BuildOutput"
    "ContentRoot" "D:\\GameBuild"
    "SetLive" "default"
    "Preview" "0"
    "Local" ""
    
    "Depots"
    {
        "YOUR_DEPOT_ID"
        {
            "FileMapping"
            {
                "LocalPath" "*"
                "DepotPath" "."
                "Recursive" "1"
            }
        }
    }
}
```

**上传构建：**
```bash
run_app_build D:\path\to\app_build.vdf
```

#### 5.2 设置分支

**主分支（default）：**
- 公开发布版本

**Beta分支：**

### 13. 常见问题和解决方案

#### 问题1：构建上传失败
**解决方案：**
- 检查网络连接
- 验证SteamCMD登录
- 检查文件权限
- 验证VDF文件格式
- 检查磁盘空间

#### 问题2：成就不解锁
**解决方案：**
- 验证Steam初始化
- 检查成就API调用
- 测试Steam测试账户
- 检查成就配置
- 查看Steam日志

#### 问题3：云存档不同步
**解决方案：**
- 验证云存档配置
- 检查文件路径
- 测试冲突解决
- 检查文件大小限制
- 验证Steam云状态

#### 问题4：低愿望单转化率
**解决方案：**
- 优化商店页面
- 改进预告片
- 更新截图
- 调整定价
- 增加营销活动

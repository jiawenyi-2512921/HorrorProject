# 预告片脚本模板

## 预告片类型指南

### 1. 公告预告片 (Announcement Trailer)
**时长:** 60-90秒
**目标:** 首次公开游戏，建立期待
**发布时机:** 游戏首次公布

### 2. 游戏玩法预告片 (Gameplay Trailer)
**时长:** 2-3分钟
**目标:** 展示核心玩法和机制
**发布时机:** 公告后1-2个月

### 3. 故事预告片 (Story Trailer)
**时长:** 90-120秒
**目标:** 展示叙事和角色
**发布时机:** 发布前3-6个月

### 4. 发布预告片 (Launch Trailer)
**时长:** 60-90秒
**目标:** 最终推动购买
**发布时机:** 发布前1-2周

### 5. 特性预告片 (Feature Trailer)
**时长:** 30-60秒
**目标:** 突出特定功能或更新
**发布时机:** 更新或DLC发布

## 公告预告片脚本模板

### 基本信息
- **预告片名称:** [游戏名] - 公告预告片
- **时长:** 75秒
- **目标受众:** 恐怖游戏爱好者、核心玩家
- **核心信息:** 游戏存在、类型、氛围、发布窗口

---

### 脚本结构

#### 第一幕：吸引注意 (0:00-0:15)
**视觉:**
- 黑屏，环境音效渐入
- 微弱的光线，模糊的环境
- 快速闪现恐怖意象
- 建立不安氛围

**音频:**
- 环境音：风声、远处的声音
- 音效：心跳、呼吸
- 音乐：低沉、紧张的音调

**文字:**
```
[无文字，纯氛围]
```

**旁白/对白:**
```
[角色声音，低语]
"它们在黑暗中等待..."
```

**镜头:**
1. 0:00-0:05 - 黑屏，声音渐入
2. 0:05-0:10 - 第一人称视角，手电筒照亮走廊
3. 0:10-0:15 - 快速切换：门、影子、眼睛

---

#### 第二幕：展示世界 (0:15-0:35)

**音频:**

- 音乐：逐渐增强

**文字:**


**镜头:**
1. 0:15-0:20 - 环境全景：废弃建筑外观
2. 0:20-0:25 - 内部探索：检查物品、阅读笔记
3. 0:25-0:30 - 解谜场景：操作机关

- 音效：威胁的声音、追逐
- 音乐：达到高潮

**文字:**

## 制作指南

### 视觉风格
- **色调:** 暗色调，高对比度
- **剪辑:** 根据节奏，紧张场景快速剪辑
- **镜头:** 第一人称为主，偶尔第三人称展示
- **特效:** 最小化，保持真实感
- **UI:** 在游戏玩法片段中隐藏或最小化

### 音频设计
- **音乐:** 原创配乐，建立情绪
- **音效:** 增强关键时刻
- **旁白:** 专业配音，神秘或紧张语气
- **混音:** 清晰的对话，平衡的音效
- **动态范围:** 利用安静和响亮的对比

### 文字和字幕
- **字体:** 清晰易读，符合游戏风格
- **大小:** 足够大以便在移动设备上阅读
- **持续时间:** 足够长以便阅读（最少3秒）
- **位置:** 通常底部居中，避免遮挡重要视觉
- **动画:** 简单淡入淡出，或符合游戏风格

### 技术规格
- **分辨率:** 1920x1080 (最低), 4K (推荐)
- **帧率:** 60fps (推荐), 30fps (可接受)
- **格式:** MP4 (H.264)
- **音频:** AAC, 立体声或5.1
- **比特率:** 视频 20-50 Mbps, 音频 320 kbps
- **长宽比:** 16:9 (标准), 也准备 1:1 和 9:16 用于社交媒体

## 平台特定版本

### YouTube版本
- **时长:** 完整版本
- **开头:** 前5秒最吸引人（跳过前广告）
- **描述:** 详细，包含链接和时间戳
- **标签:** 相关关键词
- **字幕:** 上传SRT文件

### Twitter/X版本
- **时长:** 30-45秒
- **格式:** 压缩精华版
- **文字:** 最小化（移动观看）
- **音频:** 假设静音观看，添加字幕

### Instagram版本
- **时长:** 30-60秒
- **比例:** 1:1 (Feed) 或 9:16 (Stories/Reels)
- **风格:** 快节奏，视觉冲击
- **文字:** 大而清晰

### TikTok版本
- **时长:** 15-30秒
- **比例:** 9:16 (竖屏)
- **风格:** 极快节奏，即时吸引
- **音频:** 音乐驱动

### Steam版本
- **时长:** 可以更长（2-3分钟）
- **内容:** 详细游戏玩法
- **格式:** 高质量
- **自动播放:** 前15秒无声也要吸引人

## 最佳实践

### 做
✓ 前5秒抓住注意力
✓ 展示实际游戏画面
✓ 保持节奏紧凑
✓ 突出独特卖点
✓ 包含明确的行动号召
✓ 针对不同平台优化
✓ 测试静音观看效果
✓ 包含发布日期和平台

### 不要
✗ 使用误导性画面
✗ 过度依赖过场动画
✗ 剧透关键情节
✗ 使用过时的画面
✗ 忽视音频质量
✗ 文字过小或过快
✗ 超过目标时长
✗ 忘记行动号召

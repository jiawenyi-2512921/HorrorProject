# Itch.io Release Guide
# Itch.io发布指南

## 概述

Itch.io是一个独立游戏友好的平台，以其灵活性、低门槛和社区导向而闻名。非常适合独立开发者、实验性项目和早期访问。

---

## 1. 账户设置

### 1.1 创建账户

- [ ] 访问 https://itch.io/
- [ ] 注册账户（免费）
- [ ] 验证邮箱
- [ ] 设置用户名（将成为URL的一部分）
- [ ] 完善个人资料

### 1.2 开发者设置

- [ ] 进入账户设置
- [ ] 启用"卖家账户"
- [ ] 设置支付信息
  - PayPal账户
  - Stripe账户（推荐）
- [ ] 设置税务信息
- [ ] 选择收入分成模式

### 1.3 收入分成选择

**Itch.io独特的"开放收入分成"：**
- 默认：10%给Itch.io
- 可调整：0-100%
- 推荐：10-15%（支持平台发展）
- 完全由开发者决定

## 2. 创建游戏页面

### 2.1 基本信息

#### 项目设置
- [ ] 点击"上传新项目"
- [ ] **项目URL** - 选择简短、易记的URL
  - 格式：yourusername.itch.io/gamename
- [ ] **项目标题** - 游戏名称
- [ ] **项目类型** - 可下载（Downloadable）
- [ ] **分类** - 游戏（Game）
- [ ] **发布状态** - 草稿/已发布

#### 定价模式
- [ ] **免费** - 完全免费
- [ ] **付费** - 设置固定价格
- [ ] **随心付费** - 设置建议价格，允许更多或更少
- [ ] **无最低价格** - 允许$0
- [ ] **有最低价格** - 设置最低金额

**推荐策略：**
- Demo版本：免费
- 完整版本：付费或随心付费
- 早期访问：折扣价格

## 3. 文件上传

### 3.1 准备构建

#### Windows构建
- [ ] 创建发布版本构建
- [ ] 测试构建
- [ ] 压缩为ZIP文件
- [ ] 文件命名：GameName_v1.0_Windows.zip

#### 可选平台
- [ ] Mac构建（如支持）
- [ ] Linux构建（如支持）

### 3.2 上传文件

#### 使用Web上传
- [ ] 点击"上传文件"
- [ ] 选择ZIP文件
- [ ] 等待上传完成
- [ ] 设置文件类型（可执行文件）
- [ ] 设置平台（Windows）

#### 使用Butler（推荐，用于大文件）

**安装Butler：**
```bash
# Windows
# 下载：https://itch.io/docs/butler/installing.html

# 登录
butler login

# 推送构建
butler push GameBuild yourusername/gamename:windows-stable --userversion 1.0.0
```

**Butler优势：**
- 增量更新（只上传变更）
- 更快的上传速度
- 自动版本管理
- 命令行自动化

### 4.1 定价模式选择


## 14. 常见问题

### Q: Itch.io适合商业发布吗？
A: 适合，特别是独立游戏。但Steam和Epic可能有更大的受众。

### Q: 可以同时在多个平台发布吗？
A: 可以。Itch.io没有独占要求。

### Q: 如何获得更多可见性？
A: 定期更新开发日志，参与社区，获得评分，参与活动。

### Q: 支付何时到账？
A: 根据设置，可以是即时或每月结算。

### Q: 可以更改价格吗？
A: 可以随时更改价格和定价模式。

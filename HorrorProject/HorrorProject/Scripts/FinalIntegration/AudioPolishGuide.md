# 音频打磨指南

## 音频架构

### 音频分层
```
主混音器 (Master)
├── 音乐 (Music)
│   ├── 背景音乐
│   ├── 战斗音乐
│   └── 环境音乐
├── 音效 (SFX)
│   ├── UI音效
│   ├── 环境音效
│   ├── 角色音效
│   └── 武器音效
├── 对话 (Dialogue)
│   ├── 主角对话
│   ├── NPC对话
│   └── 旁白
└── 环境音 (Ambience)
    ├── 室内环境
    ├── 室外环境
    └── 特殊环境
```

### 音量平衡
```
主音量:   0dB (基准)
音乐:     -6dB to -12dB
音效:     -3dB to -6dB
对话:     0dB to -3dB
环境音:   -12dB to -18dB
```

## 音效设计

### UI音效
```
按钮点击:
- 频率: 2000-4000Hz
- 时长: 50-100ms
- 音量: -6dB
- 特点: 清脆、简短

悬停效果:
- 频率: 1000-2000Hz
- 时长: 30-50ms
- 音量: -12dB
- 特点: 柔和、微妙

错误提示:
- 频率: 500-1000Hz
- 时长: 200-300ms
- 音量: -3dB
- 特点: 低沉、警示

成功提示:
- 频率: 3000-5000Hz
- 时长: 300-500ms
- 音量: -6dB
- 特点: 明亮、愉悦
```

### 环境音效
```
脚步声:
- 根据材质变化（木头、石头、金属、草地）
- 速度影响频率
- 添加轻微随机变化
- 3D空间定位

门声:
- 开门: 缓慢、吱呀声
- 关门: 快速、砰的一声
- 锁门: 金属碰撞声
- 添加混响

物品拾取:
- 轻物品: 高频、短促
- 重物品: 低频、沉重
- 金属物品: 清脆、回响
- 添加触觉反馈音
```

### 角色音效
```
受伤音效:
- 轻伤: 短促呻吟
- 重伤: 痛苦叫喊
- 致命伤: 长时间惨叫
- 根据伤害类型变化

攻击音效:
- 挥舞: 空气破裂声
- 击中: 撞击声 + 肉体声
- 暴击: 加强版击中声
- 连击: 节奏感强

呼吸音效:
- 正常: 平稳、轻微
- 奔跑: 急促、沉重
- 疲劳: 喘息、断续
- 恐惧: 颤抖、快速
```

### 武器音效
```
近战武器:
- 挥舞声: 破空声
- 击中声: 撞击 + 材质声
- 格挡声: 金属碰撞
- 收刀声: 摩擦声

远程武器:
- 射击声: 爆炸 + 回响
- 装弹声: 机械声
- 空弹声: 咔哒声
- 弹壳声: 金属落地

特殊武器:
- 能量武器: 科幻音效
- 魔法武器: 神秘音效
- 爆炸武器: 低频轰鸣
- 添加尾音效果
```

## 音乐设计

### 背景音乐
```
主题音乐:
- 时长: 2-3分钟
- 循环: 无缝衔接
- 情绪: 符合游戏基调
- 层次: 多轨道混音

探索音乐:
- 节奏: 缓慢、舒缓
- 音色: 环境、氛围
- 动态: 根据场景变化
- 循环: 长循环（5-10分钟）

战斗音乐:
- 节奏: 快速、紧张
- 音色: 打击乐、弦乐
- 动态: 根据战斗强度
- 过渡: 平滑进入/退出
```

### 音乐过渡
```
淡入淡出 (Crossfade):
- 时长: 2-4秒
- 曲线: 指数或对数
- 用于: 场景切换

分层混音 (Layering):
- 基础层: 始终播放
- 动态层: 根据情况添加
- 用于: 战斗强度变化

节拍同步 (Beat Matching):
- 对齐节拍
- 保持节奏连贯
- 用于: 音乐切换

垂直混音 (Vertical Remixing):
- 同时播放多轨
- 动态调整音量
- 用于: 情绪变化
```

### 自适应音乐
```
参数驱动:
- 玩家血量 → 音乐紧张度
- 敌人数量 → 音乐强度
- 场景危险度 → 音乐节奏
- 任务进度 → 音乐情绪

状态机:
- 探索状态 → 探索音乐
- 战斗状态 → 战斗音乐
- 潜行状态 → 潜行音乐
- 过场状态 → 过场音乐
```

## 3D音频

### 空间音效
```
距离衰减:
- 线性衰减: 简单但不真实
- 对数衰减: 更真实
- 自定义曲线: 最灵活
- 最大距离: 根据音效类型

方向性:
- 全向: 环境音
- 定向: 对话、音乐
- 锥形: 扬声器、广播
- 动态调整

遮挡/阻挡:
- 遮挡: 部分阻挡（低通滤波）
- 阻挡: 完全阻挡（静音）
- 实时计算
- 性能优化
```

### 混响系统
```
房间混响:
- 小房间: 短混响（0.5-1s）
- 大房间: 中混响（1-2s）
- 大厅: 长混响（2-4s）
- 室外: 无混响或极短

材质影响:
- 硬表面: 强反射
- 软表面: 弱反射
- 混合材质: 中等反射
- 动态计算

预设混响:
- 教堂: 长混响、高频衰减
- 洞穴: 长混响、低频增强
- 森林: 短混响、高频吸收
- 城市: 中混响、复杂反射
```

### 多普勒效应
```
启用条件:
- 快速移动物体
- 车辆音效
- 飞行物体
- 投射物

参数设置:
- 多普勒因子: 0.5-1.0
- 最小速度: 5 m/s
- 最大速度: 100 m/s
- 平滑过渡
```

## 对话系统

### 对话录制
```
录音标准:
- 采样率: 48kHz
- 位深度: 24bit
- 格式: WAV (无损)
- 环境: 隔音室

后期处理:
- 降噪
- 均衡
- 压缩
- 限制
```

### 对话混音
```
清晰度优化:
- 高通滤波: 去除低频噪音
- 压缩: 平衡音量
- 去齿音: 减少刺耳声
- 增强中频: 提高清晰度

空间处理:
- 近距离: 干声为主
- 中距离: 添加轻微混响
- 远距离: 低通滤波 + 混响
- 对讲机: 带通滤波 + 失真
```

### 字幕同步
```
时间轴对齐:
- 精确到帧
- 提前显示（200ms）
- 延迟消失（500ms）
- 支持多语言

文本格式:
- 字号: 可调节
- 颜色: 高对比度
- 背景: 半透明
- 位置: 可自定义
```

## 性能优化

### 音频压缩
```
音乐:
- 格式: OGG Vorbis
- 比特率: 128-192 kbps
- 质量: 高

音效:
- 格式: WAV (短音效)
- 格式: OGG (长音效)
- 比特率: 96-128 kbps
- 质量: 中高

对话:
- 格式: OGG Vorbis
- 比特率: 96-128 kbps
- 质量: 中高
```

### 音频流送
```
流送策略:
- 音乐: 始终流送
- 长音效: 流送
- 短音效: 预加载
- 对话: 按需加载

缓冲设置:
- 缓冲大小: 4096 samples
- 缓冲数量: 4
- 预加载时间: 1秒
- 优先级管理
```

### 音频池管理
```
音源池:
- 最大音源数: 32-64
- 优先级系统
- 自动回收
- 距离剔除

内存管理:
- 预加载常用音效
- 卸载未使用音效
- 内存预算: 100-200MB
- 监控工具
```

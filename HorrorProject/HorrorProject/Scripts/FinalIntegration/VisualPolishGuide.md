# 视觉打磨指南

## 光照系统

### 全局光照
```
天空光照:
- 强度: 0.5-1.5
- 颜色: 根据时间/天气
- 使用HDRI环境贴图
- 启用实时捕获

定向光 (太阳):
- 强度: 3-10
- 角度: 根据时间变化
- 阴影: 级联阴影贴图
- 光线追踪阴影（可选）
```

### 动态光照
```
点光源:
- 衰减半径: 根据光源类型
- 强度: 1000-5000 流明
- 阴影: 动态阴影
- 性能: 限制数量

聚光灯:
- 内锥角: 20-30度
- 外锥角: 40-60度
- 强度: 2000-10000 流明
- 用途: 手电筒、灯具

矩形光:
- 用于窗户、门
- 柔和阴影
- 真实光照
- 性能开销较大
```

### 光照烘焙
```
Lightmass设置:
- 质量: Production
- 光照图分辨率: 256-1024
- 间接光反弹: 3-5次
- 天空光反弹: 1次

优化技巧:
- 使用Lightmap UV
- 合理设置分辨率
- 重要区域高分辨率
- 背景区域低分辨率
```

## 材质系统

### PBR材质
```
基础参数:
- Base Color: 反照率贴图
- Metallic: 金属度 (0-1)
- Roughness: 粗糙度 (0-1)
- Normal: 法线贴图
- AO: 环境遮蔽

高级参数:
- Specular: 镜面反射
- Emissive: 自发光
- Opacity: 透明度
- Subsurface: 次表面散射
```

### 材质类型
```
金属材质:
- Metallic: 1.0
- Roughness: 0.2-0.6
- 高反射
- 无次表面散射

非金属材质:
- Metallic: 0.0
- Roughness: 0.4-0.9
- 漫反射为主
- 可能有次表面散射

玻璃材质:
- Opacity: 0.1-0.5
- Roughness: 0.0-0.2
- Refraction: 1.5
- 折射效果
```

### 材质优化
```
纹理优化:
- 使用纹理压缩
- 合理的纹理尺寸
- Mipmap生成
- 虚拟纹理（大场景）

着色器优化:
- 减少指令数
- 避免复杂计算
- 使用材质函数
- LOD材质
```

# UI打磨指南

## 布局与对齐

### 网格系统
- 使用8px或4px基础网格
- 所有元素对齐到网格
- 保持一致的间距规则
- 使用自动布局减少手动调整

### 间距规范
```
超小间距: 4px  - 紧密相关元素
小间距:   8px  - 相关元素组
中间距:   16px - 不同组之间
大间距:   24px - 主要区块
超大间距: 32px - 页面级分隔
```

### 对齐原则
- 左对齐用于文本内容
- 居中对齐用于标题和按钮
- 右对齐用于数字和操作
- 避免混合对齐方式

## 颜色系统

### 主色调
```
主色:     #1A1A2E (深蓝黑)
次色:     #16213E (深蓝)
强调色:   #E94560 (红色)
背景色:   #0F0F1E (纯黑)
```

### 文本颜色
```
主文本:   #FFFFFF (100% 不透明度)
次文本:   #FFFFFF (70% 不透明度)
禁用文本: #FFFFFF (40% 不透明度)
链接文本: #E94560
```

### 状态颜色
```
成功:     #4CAF50 (绿色)
警告:     #FFC107 (黄色)
错误:     #F44336 (红色)
信息:     #2196F3 (蓝色)
```

### 对比度要求
- 正常文本: 至少 4.5:1
- 大文本: 至少 3:1
- UI组件: 至少 3:1
- 使用工具验证对比度

## 字体系统

### 字体族
```
主字体: Roboto / Noto Sans SC (中文)
代码字体: Consolas / Source Code Pro
装饰字体: 特殊标题使用
```

### 字号规范
```
H1: 32px - 页面主标题
H2: 24px - 区块标题
H3: 20px - 小节标题
Body: 16px - 正文
Small: 14px - 辅助信息
Tiny: 12px - 标签和注释
```

### 字重使用
```
Light (300):   装饰性文本
Regular (400): 正文
Medium (500):  强调文本
Bold (700):    标题和按钮
```

### 行高规范
```
标题: 1.2 - 紧凑
正文: 1.5 - 舒适阅读
长文本: 1.6 - 更好的可读性
```

## 按钮设计

### 按钮层级
```
主按钮 (Primary):
- 背景: 强调色
- 文字: 白色
- 用于主要操作

次按钮 (Secondary):
- 背景: 透明
- 边框: 主色
- 用于次要操作

文本按钮 (Text):
- 无背景无边框
- 用于低优先级操作
```

### 按钮状态
```
Normal:
- 默认样式

Hover:
- 亮度 +10%
- 添加微妙阴影
- 过渡时间: 0.2s

Active:
- 亮度 -10%
- 轻微缩放: 0.98

Disabled:
- 不透明度: 40%
- 鼠标样式: not-allowed
```

### 按钮尺寸
```
大按钮:   高度 48px, 内边距 24px
中按钮:   高度 40px, 内边距 20px
小按钮:   高度 32px, 内边距 16px
迷你按钮: 高度 24px, 内边距 12px
```

## 图标系统

### 图标规范
- 使用统一的图标库
- 图标尺寸: 16px, 24px, 32px, 48px
- 线条粗细: 2px
- 圆角: 2px
- 保持视觉重量一致

### 图标使用
```
导航图标: 24px
按钮图标: 20px
列表图标: 16px
大图标:   48px
```

### 图标颜色
- 继承文本颜色
- 状态图标使用状态颜色
- 装饰图标可使用品牌色

## 动画效果

### 过渡动画
```
快速: 0.1s - 微交互
标准: 0.2s - 按钮、链接
中等: 0.3s - 面板、对话框
慢速: 0.5s - 页面切换
```

### 缓动函数
```
ease-out: 元素进入
ease-in: 元素退出
ease-in-out: 状态变化
cubic-bezier: 自定义曲线
```

### 动画类型
```
淡入淡出 (Fade):
- 不透明度变化
- 用于内容切换

滑动 (Slide):
- 位置变化
- 用于抽屉、菜单

缩放 (Scale):
- 大小变化
- 用于模态框

弹跳 (Bounce):
- 弹性效果
- 用于提示、通知
```

### 性能优化
- 优先使用 transform 和 opacity
- 避免动画 width/height
- 使用 will-change 提示浏览器
- 限制同时动画数量

## 表单设计

### 输入框
```
高度: 40px
内边距: 12px
边框: 1px solid
圆角: 4px
字号: 16px
```

### 输入框状态
```
Normal:
- 边框: #CCCCCC

Focus:
- 边框: 主色
- 阴影: 0 0 0 3px rgba(主色, 0.1)

Error:
- 边框: 错误色
- 显示错误信息

Disabled:
- 背景: #F5F5F5
- 不透明度: 60%
```

### 标签设计
- 标签在输入框上方
- 标签与输入框间距: 8px
- 必填标记: 红色星号
- 帮助文本: 小字号，次文本色

### 验证反馈
- 实时验证（失焦后）
- 错误信息在输入框下方
- 成功状态显示绿色勾选
- 加载状态显示旋转图标

## 卡片设计

### 卡片样式
```
背景: 白色或深色
边框: 1px solid #E0E0E0
圆角: 8px
阴影: 0 2px 8px rgba(0,0,0,0.1)
内边距: 16px
```

### 卡片层级
```
基础卡片:
- 无阴影或轻微阴影

悬浮卡片:
- 中等阴影
- hover时阴影加深

强调卡片:
- 深阴影
- 边框加粗
```

### 卡片内容
- 标题: H3字号
- 内容: Body字号
- 操作按钮: 右下角或底部
- 图片: 顶部或左侧

## 模态框设计

### 模态框结构
```
遮罩层:
- 背景: rgba(0,0,0,0.5)
- 点击关闭

对话框:
- 最大宽度: 600px
- 圆角: 8px
- 阴影: 0 4px 16px rgba(0,0,0,0.2)
```

### 模态框内容
```
头部:
- 标题 (H2)
- 关闭按钮 (右上角)
- 内边距: 24px

主体:
- 内容区域
- 内边距: 24px
- 最大高度: 60vh
- 滚动条

底部:
- 操作按钮
- 内边距: 16px 24px
- 右对齐
```

### 模态框动画
```
进入:
- 遮罩淡入 (0.2s)
- 对话框缩放+淡入 (0.3s)

退出:
- 对话框缩放+淡出 (0.2s)
- 遮罩淡出 (0.2s)
```

## 通知系统

### 通知类型
```
Toast (轻提示):
- 位置: 顶部中央或右上角
- 自动消失: 3-5秒
- 不阻塞操作

Alert (警告):
- 位置: 页面顶部
- 需要手动关闭
- 可包含操作按钮

Snackbar (底部提示):
- 位置: 底部中央
- 自动消失: 3秒
- 可包含撤销操作
```

### 通知样式
```
成功:
- 背景: 成功色
- 图标: 勾选

警告:
- 背景: 警告色
- 图标: 感叹号

错误:
- 背景: 错误色
- 图标: 叉号

信息:
- 背景: 信息色
- 图标: i
```

## 响应式设计

### 断点设置
```
手机:   < 768px
平板:   768px - 1024px
桌面:   > 1024px
大屏:   > 1440px
```

### 响应式策略
- 移动优先设计
- 流式布局
- 弹性图片
- 隐藏/显示元素
- 调整字号和间距

### 触摸优化
- 最小点击区域: 44x44px
- 增加按钮间距
- 避免悬停效果
- 使用手势操作

## 可访问性

### 键盘导航
- 所有交互元素可聚焦
- Tab顺序合理
- 显示焦点状态
- 支持快捷键

### 屏幕阅读器
- 使用语义化HTML
- 添加ARIA标签
- 图片添加alt文本
- 表单标签关联

### 颜色无障碍
- 不仅依赖颜色传达信息
- 提供文本或图标辅助
- 支持高对比度模式
- 测试色盲模式

## 性能优化

### 图片优化
- 使用WebP格式
- 响应式图片
- 懒加载
- 压缩图片

### 代码优化
- 按需加载组件
- 虚拟滚动
- 防抖和节流
- 缓存策略

### 渲染优化
- 减少重排重绘
- 使用CSS动画
- 避免强制同步布局
- 优化选择器
